# Poof Poof Pass

What are you waiting for? Pass the Poof Poofs.

## Installation

Install project dependencies first. From the project directory, run:

```bash
pip install -r requirements.txt
```

Then, install the ape plugins:

```bash
ape plugins install .
```

## Tests

To run the tests, do:

```bash
ape test
```
